# /design — تصميم UI/UX أو Component

صمّم واجهة أو component بصفة Senior UI/UX Engineer.

**الاستخدام:**
- `/design "صفحة login"` — صمم صفحة كاملة
- `/design "dropdown مع بحث"` — صمم component معين
- `/design "تحسين UX لصفحة checkout"` — حسّن UX موجود

## الخطوات:
1. افهم الـ context والمستخدم المستهدف
2. قدّم wireframe نصي أولاً
3. اكتب الكود الكامل (React + Tailwind أو CSS modules حسب المشروع)
4. اذكر: empty state، error state، loading state، mobile view
5. اذكر أي accessibility considerations مهمة
